# jinitshah.com · jinit.dev

Source for my personal website, served at [jinitshah.com](https://www.jinitshah.com) and [jinit.dev](https://jinit.dev) — a static site built with [Astro](https://astro.build).

Fonts are self-hosted and the site ships no third-party scripts or analytics. Page transitions and scroll animations use [Motion](https://motion.dev), the only non-Astro dependency.

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

The static output lands in `dist/`, with a preview available via `npm run preview`.
